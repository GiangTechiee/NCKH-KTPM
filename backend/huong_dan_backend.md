# Hướng dẫn Backend Node.js

## Cài đặt

```bash
npm install
```

## Chạy dự án

Chế độ phát triển:
```bash
npm run dev
```

Chế độ production:
```bash
npm start
```

## API hiện có

- `GET /` - Thông điệp chào mừng
- `GET /api/health` - Kiểm tra trạng thái hệ thống
