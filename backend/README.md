# Backend Node.js Project

## Cài đặt

```bash
npm install
```

## Chạy dự án

Development mode:
```bash
npm run dev
```

Production mode:
```bash
npm start
```

## API Endpoints

- `GET /` - Welcome message
- `GET /api/health` - Health check
