# Hướng dẫn Frontend React

## Cài đặt

```bash
npm install
```

## Chạy dự án

```bash
npm start
```

Ứng dụng sẽ chạy tại [http://localhost:3001](http://localhost:3001)

## Build production

```bash
npm run build
```

## Cấu trúc thư mục

```
frontend/
├── public/
│   └── index.html
├── src/
│   ├── UngDung.js
│   ├── ung_dung.css
│   ├── index.js
│   └── index.css
├── package.json
├── .gitignore
└── huong_dan_frontend.md
```

## Kết nối với Backend

Frontend đã được cấu hình proxy để kết nối với backend tại `http://localhost:3000`
